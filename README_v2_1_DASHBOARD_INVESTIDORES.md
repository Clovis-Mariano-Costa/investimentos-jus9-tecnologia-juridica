# Investimentos Jus 9 — v2.1

## Tema
Dashboard interativo ampliado, gráficos de pizza, despesas calculadas e lucro estimado.

## Alterações principais
- Criado arquivo `data/fases-investimento.json` com horizontes financeiros.
- Ampliado `dashboard.html` com gráficos de pizza e caminho de maturidade.
- Criada página `despesas-lucro.html`.
- Atualizado `script.js` com renderização de gráficos SVG interativos.
- Atualizado menu do portal de investimentos com link para despesas e lucro.

## Leitura estratégica
A lógica visual reforça o caminho: quarto → MVP real → pilotos → receita → escala → governança.

## Observação
Os valores são ilustrativos e dependem de cotações, contratos, validação operacional, revisão financeira, contábil e jurídica.
